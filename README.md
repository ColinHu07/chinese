# Chinese Translator Prototype

Structure-only scaffold for a no-API Mandarin-to-English live caption translator.

## Stages

1. `01_laptop_microphone/` - Local laptop microphone prototype using local Whisper.
2. `02_bluetooth_microphone/` - Bluetooth microphone and glasses-as-mic experiments.
3. `03_meta_wearables_mobile_bridge/` - Mobile bridge path for Meta Wearables DAT and local laptop translation.

No translator implementation has been added yet.
